# Tiny Packages

Monorepo for several tiny-* packages, each with a focus on speed and code size.

Develop with [bun](https://bun.com/).
Intended to be compatible with [deno](https://deno.com/) and [node](https://nodejs.org/en).

## Packages

* `@levischuck/tiny-qr` - Encode data into QR codes with as little javascript as possible
* `@levischuck/tiny-png` - Encode basic PNGs with as little javascript as possible
* `@levischuck/tiny-qr-svg` - Render QR codes to SVGs, again with as little javascript as possible
* `@levischuck/tiny-qr-png` - Render QR codes to PNGs, you get the idea.
